# movie-recommender system 
a content-based movie recommender built in python that recommend similar movies based on geners and metadata using cosine similarity
## Tech Stack
-python
-pandas
-numpy
-scikit-learn
## How to Run
1.install required dependencies
2.run application using python
## Configuration
Before running the project, install all required dependencies.

If an external API is used, add your own API key in the code where indicated.
The API key is not included in this repository for security reasons.
